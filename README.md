# Minimalistic Mandelbrot / Julia Set explorer

* Left click : move to pointed position
* Mouse wheel : Zoom in / out

## Algorithms

Multiple fractal algorithms are available :

* Mandelbrot
* Julia sets
* Burning ship
* Tricorn
* Celtic
